#### 环境介绍

本环境提供一套标准的Team Foundation Server 2018 环境。

| 环境名称  | 环境说明  |
| ------------ | ------------ |
| TFS 服务器  | 应用层以及数据库服务器,已装VS2015,VS2017,并配置好构建代理  |
|Linux构建代理 | 已连接至TFS服务器 |
